

# DDEV Drupal 11.3 Quick Setup Guide (Custom Docroot)

This guide provides a streamlined, step-by-step walkthrough to set up a clean installation of the latest **\*\*Drupal 11.3\*\*** using **\*\*DDEV\*\*** with a custom site name (\`plain\`) and a custom document root (\`docroot\` instead of the default \`web\`).

---

### **Step 1: Clear Everything and Start Fresh**
Open your terminal and paste these commands to clear any old configuration and start inside a clean folder named `plain`:  
```bash  
rm -rf plain  
mkdir plain  
cd plain
```

### **Step 2: Configure DDEV Upfront**

Initialize DDEV, explicitly telling it that your site name is plain and your folder layout uses docroot:

```bash  
ddev config --project-type=drupal11 --project-name=plain --docroot=docroot  
ddev start
```
### **Step 3: Get the Configuration File**

Grab the official Drupal template config file directly from the source before anything else runs:

```Bash  
curl -sL https://raw.githubusercontent.com/drupal/recommended-project/11.3.x/composer.json -o composer.json
```
### **Step 4: Swap "web" to "docroot"**

Run this command to swap every instance of web/ to docroot/ inside that configuration file:

```Bash 
sed -i.bak 's/"web\//"docroot\//g' composer.json 
rm -f composer.json.bak
```

*Note: If you are on macOS and sed throws an error, simply open the composer.json file in a text editor and manually find-and-replace all "web/" text with "docroot/".*

### **Step 5: Install Drupal & Drush**

Now that the configuration knows exactly where to put everything, run the installation. Composer will cleanly build your docroot directory:

```Bash  
ddev composer install  
ddev composer require drush/drush
```
### **Step 6: Install the Site and Open It**

Run the automatic site setup via Drush and launch it straight into your browser:

```Bash  
ddev drush site:install --account-name=admin --account-pass=admin \-y  
ddev launch $(ddev drush uli)
```
### **Credentials Reference**

* **Username:** admin  
* **Password:** admin  
* **One-time login link:** ddev drush uli
